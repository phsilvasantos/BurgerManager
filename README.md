# BurgerManager
Sistema de gestão de uma hamburgueria delivery.

## Funcionalidades
1. Anotar pedido
2. Preparar pedido
3. Embalar pedido
4. Entregar pedido
5. Comprar suprimentos
6. Contratar funcionário
7. Demitir funcionário
8. Alterar dados de funcionário
9. Remunerar funcionários
10. Emitir relatórios
